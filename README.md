# annneethu
PhonePe Project
PhonePe Transaction Analysis Dashboard

Project Overview

This project is a **data analytics dashboard built using Python and Streamlit** to analyze PhonePe digital payment transactions across India.The dashboard provides insights into **transaction trends, insurance adoption, device usage, and user engagement**.
The goal of the project is to **extract meaningful insights from PhonePe transaction data and visualize them using interactive charts and maps.**

---

# Business Case Studies Implemented

1. **Decoding Transaction Dynamics on PhonePe**

   * State-wise transaction amount
   * Quarter-wise transaction trends
   * Category-wise transaction distribution
   * Top states by transaction amount
   * State-wise transaction map (Choropleth)

2. **Insurance Penetration and Growth Potential**

   * State-wise insurance amount
   * Year-wise insurance growth
   * Quarter-wise insurance trends
   * Insurance type distribution
   * Top states by insurance usage

3. **Device Dominance and User Engagement Analysis**

   * Device-wise registered users
   * Year-wise user growth

4. **Transaction Analysis for Market Expansion**

   * Top states by transaction volume
   * District-wise transaction analysis
   * State growth opportunities

5. **User Engagement and Growth Strategy**

   * State-wise registered users
   * District-wise engagement
   * Top states by user registrations

How to Run the Project

#1. Clone the Repository

```
git clone https://github.com/yourusername/phonepe-analysis.git
```

#2.Navigate to the Project Folder

```
cd phonepe-analysis
```

#3. Install Required Libraries

```
pip install -r requirements.txt
```

#4.Run the Streamlit App

```
streamlit run main.py
```
